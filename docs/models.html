models.RuleExtractionModel module¶

class models.RuleExtractionModel.RuleExtractionModel¶

Bases : object

Modèle principal de l’application d’extraction de règles.

Il gère le chargement des ontologies, l’extraction simulée des règles, leur sauvegarde, ainsi que la récupération des classes et propriétés de l’ontologie.

charger_classes_et_proprietes()¶

Extrait les classes et propriétés RDF/OWL à partir de la dernière ontologie chargée.

Remplit les dictionnaires onto_classes et onto_properties avec les URI formatées.

charger_ontologie(path)¶

Charge une ontologie depuis un fichier local.

Paramètres:

path (str) – Chemin vers le fichier de l’ontologie.

Renvoie:

True si le fichier existe et est chargé, False sinon.

Type renvoyé:

bool
