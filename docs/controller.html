controller.RuleExtractionController module¶

class controller.RuleExtractionController.RuleExtractionController(model, view)¶

Bases : object

Contrôleur principal pour l’extraction de règles à partir d’une ontologie RDF/OWL.

Ce contrôleur relie la vue (interface utilisateur) et le modèle (données et logique métier). Il gère : - les signaux connectés à l’interface utilisateur, - le chargement et la visualisation d’ontologies, - l’extraction et la validation de règles avec AMIE3, - l’affichage de graphes RDF, - l’exécution de requêtes SPARQL, - et les mesures de qualité des règles extraites.

afficher_classes()¶

Affiche la liste des classes disponibles dans l’ontologie chargée.

afficher_page(index)¶

Change la page affichée dans le stacked widget de l’interface.

afficher_proprietes()¶

Affiche la liste des propriétés disponibles dans l’ontologie chargée.

afficher_resultats_amie3(amie_result)¶

Affiche les résultats extraits par AMIE3 dans un tableau de la page extraction.

afficher_resultats_amie3_dans_table(amie_result, table_widget)¶

Affiche les règles extraites par AMIE3 dans une table donnée.

afficher_resultats_mesure_regle(result)¶

Affiche les résultats de qualité (support, confiance, etc.) pour une seule règle dans le tableau prévu à cet effet sur la page de qualité.

afficher_resultats_mesure_regles(results)¶

Affiche dans un tableau les mesures de qualité pour un ensemble de règles.

afficher_resultats_requete(results)¶

Affiche les résultats d’une requête SPARQL dans un tableau de la page d’analyse.

charger_fichier_regles()¶

Charge un fichier contenant des règles textuelles et les affiche dans la table de comparaison.

do_afficher_graphe_ontologie()¶
do_charger_ontologie()¶

Ouvre une boîte de dialogue pour sélectionner un fichier OWL, le charge dans le modèle, et affiche une notification de confirmation ou d’erreur.

do_comparer_resultats()¶

Compare les règles extraites avec les règles chargées ou celles extraites par AMIE3. Affiche les résultats dans la page de comparaison.

do_extraire_regles()¶

Extrait une règle via le modèle et l’affiche dans l’interface utilisateur.

do_lancer_amie3()¶

Lance l’exécution de l’outil AMIE3 après conversion de l’ontologie OWL en Turtle. Vérifie les prérequis et construit la commande avant exécution. Retourne la sortie standard si l’exécution réussit.

do_lancer_amie3_avec_sauvegarde()¶

Lance AMIE3 et enregistre la sortie dans un fichier texte ou CSV selon le format choisi par l’utilisateur. Affiche les résultats extraits après l’exécution.

do_lister_regles()¶

Affiche toutes les règles extraites dans le tableau de résultats.

do_lister_regles_validees()¶

Affiche les règles validées dans le tableau de résultats.

do_mesurer_qualite_regle()¶

Mesure la qualité d’une règle saisie par l’utilisateur via l’interface, affiche les résultats dans le tableau dédié et passe à la page de résultats.

do_mesurer_qualite_regles()¶

Mesure la qualité de plusieurs règles saisies par l’utilisateur dans une zone de texte et affiche les résultats pour chaque règle dans le tableau.

do_sauvegarder_regles()¶

Ouvre une boîte de dialogue pour sauvegarder les règles validées dans un fichier CSV.

do_supprimer_regles_valides()¶

Supprime les règles validées sélectionnées et les remet dans la liste des règles extraites. Met à jour les boutons et le tableau en conséquence.

do_valider_regle()¶

Valide les règles cochées dans le tableau et les transfère dans la liste des règles validées. Supprime ces règles de la liste des règles extraites.

do_visualiser_ontologie()¶

Affiche la page de visualisation de l’ontologie si une ontologie est chargée.

do_visualiser_regles()¶

Affiche toutes les règles extraites sous forme textuelle dans l’interface.

executer_requete()¶

Exécute une requête SPARQL saisie dans l’interface sur l’ontologie chargée, et affiche les résultats dans la page d’analyse.

extraire_regles_amie3()¶

Lance AMIE3, enregistre les règles extraites dans le modèle, puis les affiche dans l’interface.

reset_view()¶

Réinitialise la taille de police des QTextEdit à 10 points sur la page actuelle.

sauvegarder_regles_extraites_fichier()¶

Sauvegarde toutes les règles extraites (y compris non validées) dans un fichier CSV.

zoom_in()¶

Agrandit la taille de police des QTextEdit sur la page actuellement affichée.

zoom_out()¶

Réduit la taille de police des QTextEdit sur la page actuellement affichée.
